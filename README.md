# Cinema-App
Is an app which allows you to manage cinema. After signing into employee account you can add movie genres, movies and it's screenings. As a customer you can choose which movie you wantch to watch and reserve a seat for it.

INSTRUCTION:
1. You need to have MS SQL Server on your computer, after building project database will be created automatically with seeded data.
